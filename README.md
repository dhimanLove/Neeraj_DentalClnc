# 🦷 Dental Clinic Website – Freelance Client Project

This is a modern, responsive dental clinic website built as a **freelance client project**.

The project focuses on:
- Clean medical branding
- Premium layout structure
- Smooth GSAP animations
- Strict responsive image handling
- Form validation with Indian mobile support
- Professional UI using shadcn & Tailwind

---

## 🚀 Tech Stack

- Vite
- React
- TypeScript
- Tailwind CSS
- shadcn-ui
- GSAP (ScrollTrigger)

---

## 🎯 Key Features

- Fully responsive hero section
- Doctor/team showcase grid
- Testimonials section
- Appointment form with:
  - Country code selector (+91 default)
  - Indian mobile validation (6–9 starting rule)
  - Email validation
  - Required field handling
- Styled dropdowns (no transparency bugs)
- Modern card system
- Controlled desktop image proportions
- Smooth scroll-triggered animations

---

## 💡 Project Goals

This project was developed to demonstrate:

- Advanced frontend layout structuring
- Real-world form validation logic
- UI refinement for medical websites
- Animation integration using GSAP
- Clean component architecture
